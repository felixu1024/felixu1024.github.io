Spring Boot之Spring Initializr搭建

开始之前

我们都知道，要新建一个Spring Boot项目，我们不管在start.spring.io还是在IDE上创建，其实都还是用的start.spring.io创建的。然后最近十九大以及之前高强度的禁止翻墙，导致现在这个时不时就会抽风一下，出现无法访问的情况，这时候就很尴尬了。后来发现，其实我们是可以自己搭建的。

搭建工程

  • 首先，我们需要下载源码
  • 在下载的工程根目录执行mvn install安装到本地仓库

  • 创建我们自己的一个Spring Boot项目，并添加刚刚打包的工程

  • 删掉我们新建项目resources下的配置文件，从源码项目中的initializr-service模块中的resources下复制配置文件到我们自己的项目中

  • 修改配置文件
