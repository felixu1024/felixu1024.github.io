一七年终总结

felixu | 发表于 2018-02-05 | 分类于 随记 |
生活不止是眼前的苟且，还有日后的苟且加枸杞

转眼2017就已经结束了，我也在这个辞旧迎新的时候来凑个热闹写一波总结。

我的新工地

几经辗转之后最终还是回了深圳，那总归还是要继续搬砖啊。这次面试什么的印象还真的不深，想着先找个差不多的坑先待着吧，所以就开始投简历了。说起来新东家的面试过程比较曲折(坑爹)，所以还是有些印象的。
本来都不打算去面试了，毕竟这公司是真的不好找，特别是导航在这一块根本是毫无作用啊。好在有hr小姐姐耐心的给我指路，在千难万阻之后，终于是找到了公司。
第一面是我现在的老大面的，聊了聊之前几个项目，介绍了一下现在的公司，整个过程轻松加愉快吧算是。
第二面是部门的老大也就是公司的副总面的，印象深刻的一个问题。你觉得你会开玩笑吗？说实话，听到这个问题当时脑子是懵逼的，黑人问号啊，这TM什么鬼啊！！！他简单解释了一下，大致就是希望同事们都外向一点，不要太沉闷。对于这个问题，我记得我的回答是，在上一家公司，我们几个程序员颠覆了公司其他部门所有人对程序员的看法，逗比、开车无所不能啊。然后稀里糊涂就过了副总面。
第三面是hr总面，说起来这一面，着实是觉得蛋疼啊，总之一句话很墨迹。。。墨迹到想狗带。
随后是什么定薪发offer，折腾了差不多一个星期吧，总算是发offer了，因为之前还挺喜欢这家公司的环境和氛围，所以当时是拿了其他的offer一直在等这家的offer，好在在我快打算入职其他公司的时候这家的offer是下来了。
入职之后觉得等待还是值得的啊，福利不错，环境氛围都不错，也不算是特别忙，有足够的时间来学习。

时间都去哪了

虽说是不忙啊，但是还是想说啊，时间都去哪了啊。。。年前定好的计划完成的可以说是不忍直视啊。

  1. 找个自己喜欢工作
    上面说了， 还是挺喜欢现在的公司。
  2. 看书计划
    年初那会计划今年要看的几本书，《java8实战》、《java核心技术卷一》、《鸟哥的私房菜》、《effective java》、《java并发编程的艺术》、《数据结构》、《算法》，现在初步看下来，也就《java8实战》、《java核心技术卷一》看完了，然后中途业务需要插进来看了《疯狂workflow》、《Docker实战》、《深入理解java虚拟机》、《Spring Cloud与Docker微服务架构实战》、《设计模式之禅》，但是还是有很多没看完的，《数据结构》、《java并发编程的艺术》看了一点，《深入理解java虚拟机》、《设计模式之禅》没看完，还有一些就木有开始看啊。算是完成了50%???
  3. GitHub活跃
    emmmm。。。这个，没坚持下来。
  4. 编码习惯
    很感谢前一家公司遇到的两位腾讯出来的大佬，跟他们在一起混了大半年，入职这边之后不久阿里发布阿里规约，第一时间是学习了一下，发现之前很多不好的习惯在之前两位大佬的指导下算是改了不少。还是有一些不好的习惯在继续改。
  5. 养成写博客(笔记)的习惯
    先是尝试在简书上写的，后来又到CSDN。但是不得不说，坚持写博客真的是件挺难的事情啊。再到后来还是决定自建吧，这样或许能多点坚持下来的理由。后面在10月份的时候就fork了一下hexo的仓库，算是自建了一个，配了个域名，一直是坚持到现在。对我而言这个东西权当是个笔记了，不敢出去误人子弟啊。

另一方面，身在金融公司，却一点不懂金融业务，着实是觉得有点蛋疼啊，还是希望18年能接触到公司其他方面都业务。

2018要做的事

生活不止眼前的枸杞，还有早睡和早起，年纪虽然是大了，生活还是要继续啊。如年初一样，再给2018年定下几个小目标吧：

  1. xxxxxxxxxx(这个放心里吧)
  2. 不管是工作学习生活坚持做总结
  3. 坚持锻炼(一周三次左右吧至少)
  4. 书单就不要了吧，主要是大学计算机基础和平常用到的技术
  5. 趁着一个人再做一些事
  6. 旅行
  7. 学习理财知识(虽然没有财，但是迟早是要改变收入组成的)

写在最后

至此，已经毕业两年半，曾经不顾一切从建筑行业踏入IT行业，虽还是一事无成，却从未后悔。
只是，初衷变了，风流飞也不再是从前的那个风流飞了，不念过去，不畏将来。接下来的日子，希望可以在技术的路上走的更远，也希望程序员都生活不是只有代码，过得多姿多彩些。
今天是我今年的最后一个工作日，谨以此文结束这个坑爹的2017，2018对这个傻逼好点啊。
